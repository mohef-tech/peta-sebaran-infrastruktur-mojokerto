# Peta Sebaran Infrastruktur - Dinas PUPR Kab. Mojokerto 🗺️

## 📝 Ringkasan Proyek
Proyek ini adalah platform berbasis web yang dikembangkan untuk memetakan dan memvisualisasikan data sebaran bangunan serta infrastruktur yang telah dibangun oleh Dinas PUPR di wilayah Kabupaten Mojokerto. Sistem ini memudahkan pemantauan aset daerah melalui tampilan peta interaktif.

---

### ✨ Fitur Utama
* **Pemetaan Interaktif:** Visualisasi titik lokasi proyek infrastruktur berdasarkan tahun anggaran (2023 - 2026+).
* **Dashboard Statistik:** Menampilkan total proyek dan ringkasan data tahunan secara real-time.
* **Filter Data:** Kemampuan untuk menyaring data proyek berdasarkan kategori dan periode pembangunan.

---

### 🛠️ Detail Teknis
* **Backend:** Node.js & Express.js.
* **Frontend:** JavaScript Framework (Vite-based) dengan integrasi Map API.
* **Database:** MySQL (Manajemen data koordinat dan detail proyek).
* **Tools:** Nodemon untuk lingkungan pengembangan yang efisien.

---

### 💡 Masalah & Solusi
* **Masalah:** Sulitnya memvisualisasikan sebaran fisik proyek pembangunan yang tersebar luas di wilayah kabupaten jika hanya menggunakan laporan tekstual.
* **Solusi:** Membangun sistem GIS sederhana yang memplot setiap data koordinat proyek ke dalam peta digital, sehingga memberikan gambaran spasial yang akurat bagi pengambil kebijakan.

---

### 🤖 Alur Kerja AI-Augmented
Dokumentasi teknis dan strategi optimasi query database pada proyek ini dikelola menggunakan **alur kerja AI-Augmented** untuk memastikan efisiensi kode dan standar dokumentasi yang profesional.
